Schedule Must Have

The schedule below has been realised to control a smal industrialization project with:

  • A Design phase based on specifications shared as main input
  • A Procurement phase to identify an external supplier that will produce some parts for qualification
  • A Test and Validation phase to confirm the part is ready for serial manufacturing

During the course of the project, Final Design: 3D models and drawing has been delayed. To complete the Start of Production milestone on time, the design activity has been split to allow partial release to anticipate some works.
